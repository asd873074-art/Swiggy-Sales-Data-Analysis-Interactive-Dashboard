# Swiggy Sales Analysis Dashboard

## Overview
This project features a comprehensive analysis of a large-scale Swiggy dataset containing nearly **1.97 Lakh orders** using **Microsoft Excel**. The project transforms raw transactional data into an interactive visual dashboard to track business performance, customer satisfaction, and regional sales distribution.

## Objective
The goal of this project was to analyze Swiggy's sales operations across India to identify high-performing regions, understand customer spending habits, and monitor service quality through ratings. This helps in identifying which markets are driving growth and where service improvements are needed.

## Dashboard Features
* **KPI Scorecard:** Instant visibility of **Total Sales (₹5.3Cr)**, **Total Orders (1.97L)**, **Average Rating (4.34)**, and **Average Order Value (₹268)**.
* **Geospatial Sales Analysis:** Bar charts identifying the top revenue-generating states.
* **Customer Satisfaction Tracking:** Rating distribution by City and State to pinpoint service excellence or gaps.
* **Interactive Filtering:** Dynamic **Slicers** for State and City, allowing users to drill down into specific regional data instantly.

## Tools Used
* **Microsoft Excel:** Core tool for data cleaning and visualization.
* **Pivot Tables & Charts:** For efficient data aggregation and summary.
* **Excel Slicers:** To create an interactive user experience and real-time filtering.
* **Data Cleaning:** Processing a real-world dataset of ~200,000 rows.

## Key Insights
* **Revenue Leaders:** **Karnataka** and **Maharashtra** emerged as the highest revenue-contributing states.
* **Customer Loyalty:** The platform maintains a high average rating of **4.34/5.0**, indicating strong service consistency.
* **Market Benchmarking:** Established an **Average Order Value (AOV) of ₹268** across the dataset, providing a clear benchmark for typical consumer behavior.
* **Regional Performance:** Identified top-rated cities, providing a roadmap for service expansion.
